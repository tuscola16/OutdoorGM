## Full Description (both stores)

Outdoor GM turns any outdoor space into a real-time GPS game. Whether you're running a scavenger hunt, an adventure race, or a team-building exercise, Outdoor GM gives the Game Master full visibility while players explore.

HOW IT WORKS

Game Masters create a game and place checkpoints on a map. Players join with a simple 6-character code. As players move, their GPS positions update on the GM's live map in real time. When a player reaches a checkpoint, the GM gets an instant push notification.

FOR GAME MASTERS
- Create games and share invite codes
- Place checkpoints anywhere on the map with custom detection radii
- Watch all players move in real time on a live satellite map
- Get instant push notifications when players reach checkpoints
- Manage players: promote co-GMs, remove players
- End the game when you're done

FOR PLAYERS
- Join with a code — no account setup beyond phone verification
- See your own position on a mini-map
- Background GPS tracking works even with the app closed
- Get notified when the game ends

PRIVACY FIRST
Players can only see their own location. Only Game Masters see the full map. Location sharing stops the moment you leave a game. Delete your account anytime from the profile screen.

PERFECT FOR
- Scavenger hunts and treasure hunts
- Adventure races and orienteering
- Team building events
- Field games and outdoor education
- Any activity where a coordinator needs to track participants
